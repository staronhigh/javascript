javascript 1회차

기본문법

  1. html내 head, body안에 들어갈수 있다. (body요소 가장아래가 좋은위치)
  2. 구문은 세미콜론(;)으로 끝나야 한다.
  3. 주석은 두가지 한줄주석(//) , 여러줄주석(/**/)

네이밍규칙

  1. 대소문자를 구분해야 한다.
  2. 키워드를 사용하면 안된다. (키워드 : http://poiemaweb.com/js-syntax-basics#9-키워드-keyword)
  3. 숫자로 시작하면 안된다.
  4. 특수문자는 _과 $만 허용한다.
  5. 공백문자를 포함할 수 없다.

변수 (variable)

javascript 에서 변수는 data를 저장, 참조하기 위해 사용된다. 식별자라고도 불린다. 변수에는 함수,객체,배열 모든것들을 담을수있다. 고정되지 않은 값을 가지는 공간이라 생각하면 된다.

        
            var name; // 변수를 선언하고 초기화한다.
            name = "김혜진";  // 변수에 값을 할당한다.

자료형 (Data type)

기본자료형과 / object(객체형,참조형) 이 있다. 아래는 기본자료형의 5가지이다. (ECMAScript 6에서 기본자료형 symbol이 추가됨)

String

''작은따옴표 또는 ""큰따옴표를 사용하여 텍스트 데이터를 나타낸다.

Number

다른 프로그래밍 언어의 경우 보통 정수형과 실수형을 구분하지만, 자바스크립트는 하나의 숫자 자료형으로 표현한다.

boolean

논리적 요소를 나타내며, ture, false 두가지 값을 가진다. 

    
        var foo = true;
        var bar = false;

undefined

선언은 되었지만 할당된적이 없는 변수에 접근하거나 존재하지 않는 객체 프로퍼티에 접근할 경우 반환된다

null

값이 없음을 명시

object(객체형,참조형)

배열,함수,날짜,정규식

연산자

산술연산자

+, -, *, /, %

증감연산자

num++, num--, ++num, --num
변수의 값을 하나씩 증가시키거나 감소시킴

대입연산자

=, +=, -=, *=, /=, %= 연산자 오른쪽에 있는 값을 연산자 왼쪽에 대입

비교연산자

==, ===, !=, !==, > , < , >=, <=, ?(삼항연산자)

논리연산자

&&(and), ||(or), !(not) 불린값과 불린값을 연산할때 사용

조건문 (Conditional Statements)

조건식의 참, 거짓을 분별하여 참일 때와 거짓일 때의 실행문을 달리하는 제어문이다

if문

주어진 조건식을 평가하여 참, 거짓을 구별한다.

    
        if (조건식){
            조건식이 참일때 실행
        } else {
            조건식이 거짓일때 실행
        }

switch문

switch 변수의 값과 일치되는 case문으로 실행순서가 이동하고
일치되는 case문이 없을 경우 실행순서는 default로 이동한다.

break 키워드는 switch 구문에서 탈출하는 역할.
break 가 없다면 실행순서는 다음 case 문으로 이동

삼항연산자

연산자에 속하지만 조건에따라 프로그램진행을 변화시킬수 있다.


        (조건문) ? 참일때 실행 : 거짓일때 실행

반복문 (Loop)

조건식의 참, 거짓을 분별하여 참일 경우 실행문을 실행하며 조건식이 거짓일때까지 반복됨.
3가지 반복문이 있음 for, while, do while

for 문

정해진 횟수만큼 반복 실행할 때 사용

    
        for(초기문; 조건문; 증감문;){
            실행문
        }

while문

반복횟수를 알지 못할 때 사용, 

    
        while (조건문){
            실행문
        }

do while문

while문과 유사하나 최소 1회 이상 실행

    
        do {
            실행문
        } while (조건문);

break

반복문을 탈출할 때 사용하는 키워드

continue

continue문 이후의 구문은 실행 생략하고 반복문의 조건검사 위치로 이동
